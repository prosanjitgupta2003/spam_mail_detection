# Spam Email Detection using Decision Tree

## Description
This project detects whether an email/message is spam or not using a Decision Tree Machine Learning model.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Natural Language Processing (NLP)

## Model Used
- Decision Tree Classifier


## Preprocessing Steps
- Text cleaning
- Tokenization
- Removal of stopwords
- Vectorization (e.g., CountVectorizer / TF-IDF)

## Evaluation Metric
- Accuracy Score,Precision, Recall, and F1-score Calculation:

## Results
- Accuracy: 92.18%

## Features of the Project
- Text preprocessing using NLP
- Feature extraction from text data
- Model training and evaluation
- Spam classification system

## Conclusion
The Decision Tree model achieved an accuracy of 92.18% in classifying spam and non-spam messages, showing good performance on the dataset.
